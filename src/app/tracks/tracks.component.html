Beginner Track
William Eiffert
Assistant Instructor for Beginner
About
The beginner track is the perfect place for novices of all majors and experience levels to get a basic understanding of how the web works and how to build simple, beautiful websites.
Topics and Learning Objectives
Learn how web works, HTML & CSS, JavaScript, ES5 ES2015, Intro to Advanced Web Topics such as Nodejs and AngularJS.
Meets: Every Tuesday at 6PM in PHYS 116 starting January 30th.
Advanced Track
Maneesh Tewani
Lead Instructor
About
The advanced track is built for people wanting to gain a deeper understanding of more advanced web technologies needed to build more extensive webpages.
Topics and Learning Objectives
Node.js, Build Systems, Modules, TypeScript, Angular CLI, Advanced Angular 2 coverage, Redux/Immutable State, AngularFire 2
Meets: Every Wednesday at 6PM in PHYS 116 starting January 31st.
